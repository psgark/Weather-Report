## âœ¨ Features

*   **Real-Time Weather Data**: Fetches up-to-date weather information including temperature, humidity, wind speed, and "feels like" temperature.
*   **Search Functionality**: Allows users to search for weather conditions in any city globally.
*   **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices, ensuring a seamless experience across all screen sizes.
*   **Modern UI/UX**:
    *   **Glassmorphism Effect**: Features a sleek, translucent card design with backdrop filtering.
    *   **Dynamic Backgrounds**: Visually appealing gradients.
    *   **Smooth Animations**: Loading spinners and fade-in effects for a polished feel.
*   **Error Handling**: User-friendly error messages when a city is not found or if there are network issues.
